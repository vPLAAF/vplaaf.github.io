# 1. 政策目的

本《隐私政策》描述虚拟中国人民解放军空军（virtual People’s Liberation Army Air Force，简称 **vPLAAF**）如何依据 **欧盟《通用数据保护条例》（GDPR）** 收集、使用、存储与保护个人数据（GDPR 原文可在 https://gdpr-info.eu/ 查阅）。

vPLAAF 是在 VATSIM 网络上运营的非营利虚拟航空组织。
本组织仅收集用于会员管理与运行安全所必需的数据。

------

# 2. 适用范围

本政策适用于：

- 所有现役 vPLAAF 成员
- 所有前成员（在允许的保存期限内其数据仍被存储者）
- 所有申请人及候选成员
- 任何与 vPLAAF 系统交互的个人（网站、Discord、训练平台等）

------

# 3. 处理数据的合法基础

vPLAAF 在以下法律基础上处理个人数据：

- **同意权（GDPR 第 6(1)(a) 条）**
   即当你申请加入或自愿提交信息时授予的同意。
- **合法利益（GDPR 第 6(1)(f) 条）**
   用于身份验证、维护运行安全、支持组织运作、遵守 VSOA 政策等。

成员可随时撤回同意权。

------

# 4. 我们收集的数据

vPLAAF 仅收集用于身份验证、训练及参与运行所需的最少数据。

### 4.1 个人数据

- 姓名（或经申请后使用“名 + 姓氏首字母”）
- 电子邮箱地址
- VATSIM CID
- Discord 用户名 / Discord 用户 ID
- 培训记录（时长、考试记录、资质状态）
- vPLAAF 呼号与飞行活动（基于 VATSIM 的公开数据）

### 4.2 不会收集的数据

为最小化风险，vPLAAF**不会**收集以下内容：

- 电话号码
- 政府身份证号码
- 居住地址
- 面部或生物识别信息
- 支付或金融信息
- GDPR 第 9 条定义的任何敏感数据类别

vPLAAF 不进行画像（profiling）或自动化决策处理。

------

# 5. 数据的收集方式

数据主要通过以下方式收集：

- 会员申请或注册表单
- 成员自行提交的信息
- 公开可访问的 VATSIM API 数据
- Discord 服务器成员信息

访问网站的访客可匿名浏览公开内容。

------

# 6. 数据处理的目的

### 6.1 会员管理

- 身份验证
- 名册维护
- 与成员沟通
- 记录资格与培训进展

### 6.2 运行需求

- 参与符合 VSOA 的各类活动
- 活动安排与协调
- 培训项目记录以满足审计需求

### 6.3 通讯用途

- 活动公告
- 安全通知
- 行政更新

### 6.4 Cookies（网站相关）

如使用 Cookies，其用途仅包括：

- 维持登录状态
- 保存用户偏好

不使用广告或跟踪类 Cookies。

------

# 7. 数据共享与第三方

vPLAAF **不会出售、交易或商业共享**个人数据。

数据仅在以下范围共享：

### 7.1 与 VATSIM 或 VSOA 管理机构

仅在以下情况下提供：

- 合规性验证
- 安全审查
- 组织审计需求

### 7.2 服务提供商

限于用于组织运行的平台，例如：

- Google Forms
- 网站托管服务
- Discord

这些服务仅按指示处理数据，并提供与 GDPR 等效的保护。

### 7.3 公共信息要求

为遵守 VSOA 政策，vPLAAF 可能公开以下信息：

- 姓名
- VATSIM CID
- 飞行员状态（活跃 / 不活跃）

邮箱及内部记录 **绝不会公开**。

------

# 8. 跨境数据传输

vPLAAF 使用的云服务可能将数据存储在 EU/EEA 以外地区，并遵守GDPR 第 V 章要求。

提交数据即表示成员同意在以下前提下进行国际传输：

- 具备充分保障措施
- 必要时使用标准合同条款（SCC）
- 服务提供商提供与 GDPR 等效的保护

vPLAAF 不会与任何政府或商业实体共享数据。

------

# 9. 数据保留期限

vPLAAF 仅在必要时限内保留个人数据：

- 现役成员：在会员期间保留
- 前成员：最长保留 **12 个月** 以维护组织完整性
- 培训日志：最长保留 **36 个月** 以满足审计需求

期限届满后，数据会被安全删除。

成员可随时申请提前删除。

------

# 10. 数据主体权利

根据 GDPR 第 III 章，你拥有：

- **被告知权**
- **访问个人数据权**
- **更正权**
- **删除权（被遗忘权）**
- **限制处理权**
- **撤回同意权**
- **数据可携带权**

删除数据的申请会导致 vPLAAF 会员资格终止。

如需行使这些权利，请联络数据控制者。

# 12. 安全措施

vPLAAF 实施合理的技术与组织安全措施，包括：

- HTTPS 加密网站
- Discord 基于权限的访问
- 安全的云存储
- 仅限必要管理员访问个人数据
- 定期权限审查

虽然没有系统能做到绝对安全，但 vPLAAF 采取行业标准的保护方法。
